# Me-GPT: AI Portfolio (Mobile Built)

This is a Python-based interactive portfolio built entirely on an Android phone using Pydroid 3.
It allows recruiters to "chat" with my profile in a terminal-style interface.

## Features
- Interactive CLI (Command Line Interface)
- Displays Education, Skills, and Projects dynamically.
- "Hacker-style" typing animations.

## Tech Stack
- Python
- Developed on Android
